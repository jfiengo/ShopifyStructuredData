How to Use
Finding Product Handle

The product handle is the last part of your product URL. For example:

https://your-store.myshopify.com/products/product-name

In this case, the product handle is product-name

What's Analyzed
  • Existing JSON-LD schemas
  • Microdata markup
  • RDFa annotations
  • Schema completeness
  • Google Rich Results compatibility
Analysis Types

Basic Analysis: Overview of existing structured data
Detailed Analysis: Comprehensive breakdown of all schemas
Google Check: Validation against Google Rich Results requirements

What to Look For
Good Signs:
  • JSON-LD schemas present
  • Product schema with required fields
  • Organization schema
  • Breadcrumb navigation
Issues to Address:
  • Missing required fields
  • No structured data
  • Invalid schema syntax
  • Missing review data
